Programming Fundamentals:
--------------------------
1. keywords
Reserved words in any programming lang ,  keywords are written in lowercase.
Every keyword will have reserved amount of memory allocated to it.

Ex: int , short , boolean , if , else , for , try , catch , private , class , void

50+ keywords in java


Primitive Data types:
---------------------
Reserved data types who's memory amount is fixed.

byte,short,char, int ,long ,float , double ,boolean

Non-Primitive / Reference Data types:
String , Object , Student , Employee , Customer 


Identifiers:
Any predefined or user-defined name is known as Identifier.

Ex: class name , var name , package , method method , constructor etc..


Literals:
Any constant value is known as Literal

int  x = 10;  // 10 integer literal 

String  name = "javeed";  // "javeed" String literal

boolean  b1 = true; // b1 is boolean literal

String  city = null;
